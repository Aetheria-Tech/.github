# Aetheria

이 레포지토리는 **Aetheria** 내 모든 공개 GitHub 레포지토리에 대한 표준 설정과 가이드라인을 중앙 관리합니다.

---

## 1. 💡 이 레포지토리에 무엇이 포함되어 있나요?

이 레포지토리의 파일들은 조직의 모든 공개 레포지토리에 상속되어 사용됩니다.

* **[✅ 기여 가이드]**: `CONTRIBUTING.md`
* **[🤝 행동 강령]**: `CODE_OF_CONDUCT.md`
* **[🚨 보안 보고]**: `SECURITY.md`
* **[📝 이슈 템플릿]**: `ISSUE_TEMPLATE/`
* **[👀 PR 템플릿]**: `PULL_REQUEST_TEMPLATE.md`

## 2. 🚀 기여자 시작하기 (For Contributors)

코드를 기여하기 전에 다음 문서를 **반드시** 확인해 주세요.

1.  **[기여 가이드](/CONTRIBUTING.md)**: PR 생성 방법, 커밋 메시지 규칙 등을 확인합니다.
2.  **[행동 강령](/CODE_OF_CONDUCT.md)**: 우리 팀의 기본적인 협업 문화를 이해합니다.

## 3. 🛠️ 관리자/유지보수 가이드 (For Maintainers)

### 라벨 동기화
조직 라벨을 업데이트하려면, `label-sync-workflow.yml` GitHub Actions를 수동으로 실행하세요.

### 새 레포지토리 생성
새로운 레포지토리를 만들 때 `master` 브랜치 보호 규칙을 설정했는지 확인하세요.
